* Transiciones en CSS:

Hasta ahora hemos visto que cuando tenemos un cambio de estado, como por ejemplo pasar el ratón por encima de un link, este cambio se produce de forma automática. Con la propiedad transition podemos añadir una transición a los cambios de estado, lo cual puede ser muy útil para transmitir interacción.

Hover se usa por ejempo para cambiar de color al pasar por encima.

Tiene tres valores: / transition: duracion propiedad retraso;
